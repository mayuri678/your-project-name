JWT Authentication Demo

Login Page Profile Page

Implementation Overview

This demo includes a complete JWT authentication system with:

  • Login Form: Email/password authentication
  • JWT Token Storage: Tokens stored in localStorage
  • Profile Page: Fetches user data using Authorization Bearer token
  • Mock API: Simulates backend authentication

  • components/jwt-profile.component.ts - Profile display component

How It Works:

  1. User enters email/password on login form
  2. Service calls mock API to authenticate
  3. On success, JWT token is stored in localStorage
  4. Profile page uses stored token to fetch user data
  5. Token is sent as Authorization Bearer header
  6. Logout clears the token from localStorage
