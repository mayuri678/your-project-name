# Server Start Kara

## Problem
"This site can't be reached" - Server chalu nahi ahe

## Solution

### Terminal madhe command run kara:

```bash
ng serve
```

### Ani browser madhe ja:
```
http://localhost:4200
```

### Server chalu ahe ka check kara:
- Terminal madhe "Compiled successfully" disel
- Browser madhe page load hoil

---

## Commands

**Start Server:**
```bash
ng serve
```

**Stop Server:**
`Ctrl + C` terminal madhe

**Different Port:**
```bash
ng serve --port 4300
```
